# 🏠 Texas Real Estate Market Analysis

## 📝 Project Overview
An Exploratory Data Analysis (EDA) of the Texas real estate market from 2010 to 2014. The project explores historical sales trends, median prices, volumes, and inventory across various Texan cities.

## 🎯 Key Insights
* **Price Dynamics:** Cities like Bryan–College Station show higher median prices, while Wichita Falls is the most affordable.
* **Seasonality:** Consistent sales peaks during mid-year months (May-July).
* **Strategic Recommendations:** Provided actionable advice on optimal investment timing and advertising budget optimization based on conversion rates (Ad effectiveness).

## 🛠️ Technologies & Libraries
* R (RMarkdown for HTML report generation)
* Tidyverse, Lubridate, Skimr, Ggplot2
